== SIMULADOR DE VOZ CON VOSK ==

1. Verifica que el servidor VOSK esté corriendo en:
   ws://localhost:2700

2. Asegúrate de que el archivo sonidoPrueba.wav exista en:
   ..\DatosPrueba\Audio\sonidoPrueba.wav

3. Ejecuta este archivo:
   simular_voz.bat

Esto simulará el envío del audio por WebSocket y activará la transcripción en los campos del sistema INCORP_domus.
